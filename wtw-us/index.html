Welcome to the Shelf.io Migration & Style Override Tool

Summary:
1. This script migrates articles from a source library to a destination library.
2. It updates article titles by replacing "Copy of [Existing Library Name]" with the new Client Name.
3. It discovers and overrides styles in articles based on the existing styles.
4. It updates links in articles to point to the new gem IDs after copying the articles.
5. It requires a Content Block ID input for the Article Headers to be updated.
6. Creates new user groups and automatically grants permissions to the new Library.
